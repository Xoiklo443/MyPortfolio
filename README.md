# Personal Portfolio Website

A responsive personal portfolio website built using HTML5 and CSS3. This project showcases my background, technical skills, projects, resume, blog articles, and contact information through a clean, modern, and user-friendly interface.

---

## Overview

This portfolio serves as a central place to present my skills, experience, and projects in Information Technology and Cybersecurity. The website follows a responsive card-based design that works across desktops, tablets, and mobile devices.

---

## Features

- Responsive design for all screen sizes
- Modern card-based layout
- Sticky navigation bar
- About Me section
- Skills showcase
- Projects section
- Embedded resume viewer
- Contact form
- Blog section
- Social media links
- Smooth hover effects and transitions
- Clean and maintainable CSS structure

---

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- CSS Variables
- Responsive Design (Media Queries)
